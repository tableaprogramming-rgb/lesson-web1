üé® What is CSS?

üí°
Definition

CSS (Cascading Style Sheets) describes *how* HTML content looks. HTML is structure, CSS is presentation.

üìù
Basic Syntax

Every rule follows: selector { property: value; }

üìÇ
Three Ways to Add CSS

Inline (not recommended), internal in <style>, or external .css files (best).

üåä
Cascading

Later rules override earlier ones. Specificity determines which rule wins when conflicts exist.

HTML Code

style.html 
<head>
  <link rel="stylesheet"
        href="styles.css">
  <style>
    p { color: navy; }
  </style>
</head>
<body>
  <h1>Hello</h1>
  <p style="color: red;">
    Inline style
  </p>
</body>

‚öñÔ∏è Specificity & Cascade

üèÜ
Specificity Points

Element = 1pt, Class = 10pts, ID = 100pts, Inline = 1000pts. Higher specificity wins.

üìä
Calculating Specificity

.article p { } = 11pts (1 class + 1 element). #main p { } = 101pts (1 ID + 1 element).

‚ö†Ô∏è
Best Practices

Use classes for styling (flexible), avoid IDs (rigid), avoid inline styles, avoid !important.

üåä
Cascade

CSS rules flow top-to-bottom. Later rules override earlier ones *if specificity is equal*.

Example Code

specificity.css 
/* 1 point */
p { color: black; }

/* 11 points - WINS */
.article p {
  color: navy;
}

/* 101 points - HIGHEST */
#main p {
  color: darkblue;
}

/* 1000 points - BAD! */
p {
  color: red !important;
}

üì¶ The Box Model

üéÅ
Four Layers

Content (innermost) ‚Üí Padding ‚Üí Border ‚Üí Margin (outermost). Every HTML element is a box.

üìê
Margin

Space *outside* the border (transparent). Creates distance between elements.

üìè
Padding

Space *inside* the border. Creates breathing room for content.

‚öôÔ∏è
box-sizing: border-box

Include padding/border in width/height for predictable sizing. Apply globally with * { }.

CSS Code

box-model.css 
* {
  box-sizing: border-box;
}

.box {
  width: 200px;
  margin: 20px;
  border: 2px solid
          #999;
  padding: 15px;
  background:
    lightblue;
}

/* Total width:
   200px (includes
   padding & border) */

üé® Colors & Typography

üåà
Color Formats

Hex #FF5733, RGB rgb(255, 87, 51), HSL hsl(9, 100%, 64%), or named colors.

üî§
Font Family

System fonts (Arial) or web fonts (Google Fonts). Always provide fallback chain: 'Inter', sans-serif.

üìè
Font Properties

font-size, font-weight, line-height. Use 16px baseline, 1.6 line-height for readability.

üñºÔ∏è
Backgrounds

Solid color, image, or gradient. Use background-size: cover, background-position: center.

CSS Code

style.css 
body {
  font-family: 'Inter',
               sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
}

h1 {
  color: #FF5733;
  font-size: 2.5rem;
  font-weight: 700;
}

.highlight {
  background:
    linear-gradient(
      135deg,
      #667eea,
      #764ba2
    );
}

üñºÔ∏è Backgrounds & Visual Effects

üé®
Background Properties

background-color, background-image, background-size, background-position, background-repeat.

üìê
Background Size

cover fills area (may crop), contain fits entire image, or specific pixel values.

üåà
Gradients

Linear: linear-gradient(90deg, color1, color2). Radial: radial-gradient(circle, ...). No image needed.

üí°
Performance Tips

Optimize images (compress, use appropriate format). Gradients are lightweight. Use appropriate file sizes.

CSS Code

backgrounds.css 
/* Solid color */
.box1 {
  background-color:
    #F0F0F0;
}

/* Linear gradient */
.box2 {
  background:
    linear-gradient(
      135deg,
      #667eea 0%,
      #764ba2 100%
    );
}

/* Radial gradient */
.box3 {
  background:
    radial-gradient(
      circle,
      yellow,
      purple
    );
}
