üìç Positioning: Five Methods

1Ô∏è‚É£
Static (Default)

Elements flow normally. Top/left/right/bottom properties ignored. No z-index effect.

2Ô∏è‚É£
Relative

Positioned relative to its normal position. Element still occupies original space. Use for fine-tuning.

3Ô∏è‚É£
Absolute

Positioned relative to nearest positioned ancestor. Removed from document flow. Use for overlays, tooltips.

4Ô∏è‚É£
Fixed

Positioned relative to viewport. Doesn't scroll. Use for sticky headers, notifications.

5Ô∏è‚É£
Sticky

Hybrid of relative and fixed. Sticks when scrolling past threshold. Use for sticky headers, table headers.

CSS Code

positioning.css 
/* Relative positioning */
.box-relative {
  position: relative;
  top: 10px;
  left: 20px;
}

/* Absolute positioning */
.container {
  position: relative;
}
.box-absolute {
  position: absolute;
  top: 50px;
  left: 30px;
}

/* Fixed positioning */
.header {
  position: fixed;
  top: 0;
  z-index: 100;
}

üß© Flexbox: One-Dimensional Layouts

üì¶
Flex Container

Parent with display: flex. Direct children become flex items automatically.

üîÄ
Main & Cross Axis

Main axis: default horizontal. Cross axis: perpendicular (vertical). Use flex-direction to change.

üìç
justify-content

Aligns items along main axis. Values: flex-start, center, space-between, space-around.

üìè
align-items

Aligns items along cross axis. Values: flex-start, center, stretch, baseline.

CSS Code

flexbox.css 
/* Flex container */
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 15px;
  height: 100px;
}

/* Flex item */
.flex-item {
  background: indigo;
  padding: 10px 20px;
  border-radius: 6px;
  flex: 1;
}

üèóÔ∏è CSS Grid: Two-Dimensional Layouts

üéØ
Grid Container

Parent with display: grid. Direct children become grid items. Handles rows AND columns.

üìê
Grid Template Columns

grid-template-columns: 200px 1fr 200px creates 3 columns. Use fr for flexible units.

üìè
Grid Template Rows

Define row heights. Example: grid-template-rows: 100px auto 50px

üî≤
Grid Gap

gap: 20px creates space between all items. Shorthand for column-gap and row-gap.

CSS Code

grid.css 
/* 3-column grid */
.grid {
  display: grid;
  grid-template-columns:
    200px 1fr 200px;
  grid-template-rows:
    100px auto 50px;
  gap: 15px;
}

/* Grid item spanning */
.header {
  grid-column: 1 / -1;
}

.sidebar {
  grid-column: 1;
}

.content {
  grid-column: 2;
}

üì± Responsive Design & Media Queries

üéØ
Mobile-First Approach

Start with mobile layout, then enhance for larger screens. Use min-width media queries.

üìê
Common Breakpoints

Mobile: < 768px | Tablet: 768px-1024px | Desktop: > 1024px. Adjust based on design.

üîç
Media Query Syntax

@media (min-width: 768px) { ... } applies styles only when viewport ‚â• 768px.

‚öôÔ∏è
Flexible Sizing

Use percentages, max-width, flex, and fr units. Avoid fixed pixel layouts.

CSS Code

responsive.css 
/* Mobile first */
.container {
  display: grid;
  grid-template-columns:
    1fr;
  padding: 1rem;
}

/* Tablet and up */
@media (min-width: 768px) {
  .container {
    grid-template-columns:
      1fr 250px;
    padding: 2rem;
  }
}

/* Desktop and up */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}
