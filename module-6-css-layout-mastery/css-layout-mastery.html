üìç Positioning: Five Methods

1Ô∏è‚É£
Static (Default)

Elements flow normally. Top/left/right/bottom properties ignored. No z-index effect.

2Ô∏è‚É£
Relative

Positioned relative to its normal position. Element still occupies original space. Use for fine-tuning.

3Ô∏è‚É£
Absolute

Positioned relative to nearest positioned ancestor. Removed from document flow. Use for overlays, tooltips.

4Ô∏è‚É£
Fixed

Positioned relative to viewport. Doesn't scroll. Use for sticky headers, notifications.

5Ô∏è‚É£
Sticky

Hybrid of relative and fixed. Sticks when scrolling past threshold. Use for sticky headers, table headers.

CSS Code

positioning.css 
/* Relative positioning */
.box-relative {
  position: relative;
  top: 10px;
  left: 20px;
}

/* Absolute positioning */
.container {
  position: relative;
}
.box-absolute {
  position: absolute;
  top: 50px;
  left: 30px;
}

/* Fixed positioning */
.header {
  position: fixed;
  top: 0;
  z-index: 100;
}

üß© Flexbox: One-Dimensional Layouts

üì¶
Flex Container

Parent with display: flex. Direct children become flex items automatically.

üîÄ
Main & Cross Axis

Main axis: default horizontal. Cross axis: perpendicular (vertical). Use flex-direction to change.

üìç
justify-content

Aligns items along main axis. Values: flex-start, center, space-between, space-around.

üìè
align-items

Aligns items along cross axis. Values: flex-start, center, stretch, baseline.

CSS Code

flexbox.css 
/* Flex container */
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 15px;
  height: 100px;
}

/* Flex item */
.flex-item {
  background: indigo;
  padding: 10px 20px;
  border-radius: 6px;
  flex: 1;
}
