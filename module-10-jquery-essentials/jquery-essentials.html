$ jQuery Syntax & Basics

$
The $ Function

$() selects elements. $(".class"), $("#id"), $("p"). Returns jQuery object with chainable methods.

üîó
Method Chaining

Chain methods together: $("p").text("Hello").css("color", "red").addClass("highlight")

üìã
Document Ready

$(document).ready(fn) or $(fn). Waits for page load before running jQuery code.

‚öôÔ∏è
jQuery vs JavaScript

jQuery: $("p").text("Hello"). JS: document.querySelector("p").textContent = "Hello"

jQuery

jquery.js 
$(document).ready(fn(){
  // Select
  $(".highlight")
    .text("Hello")
    .css("color", "red")
    .addClass("active");

  // Event
  $("button")
    .click(function() {
      alert("Clicked!");
    });

  // Effects
  $("p").fadeIn();
  $(".box").slideToggle();
});

Modern JavaScript

modern.js 
document.addEventListener(
  "DOMContentLoaded",
  () => {
    // Select
    const el =
      document.querySelector(
        ".highlight"
      );
    el.textContent = "Hello";
    el.style.color = "red";
    el.classList.add(
      "active"
    );

    // Event
    document.querySelector(
      "button"
    ).addEventListener(
      "click", () =>
        alert("Clicked!")
    );
  }
);

    Event Methods

    $("button").click(fn), .hover(fn1, fn2), .submit(fn), .on("event", fn)

    üåê
    AJAX Requests

    $.get(url, fn), $.post(url, data, fn), $.ajax({...}). Fetch data without refresh.

    üîÑ
    Deferred/Promises

    jQuery AJAX returns promise-like object. .done(), .fail(), .always() for callbacks.

    Modern Alternative

    Fetch API replaces jQuery AJAX. Promises/async-await more standard now.

    ‚úÖ Use Modern JS
