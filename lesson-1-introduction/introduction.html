ï¿½ The History of HTML

1989 Tim Berners-Lee proposes a global hypertext system at CERN. [3]
1991 HTML 1.0: The first 18 tags are published, defining the basic web. [2][7]
1995 HTML 2.0: The first official standard for the World Wide Web.
1999 HTML 4.01: A stable version that powered the web for a decade.
2004 WHATWG forms to push HTML beyond static documents.
2014 HTML5 Finalized: Massive update with Video, Canvas, and Semantic tags. [2]
Today HTML Living Standard: Continuously updated version we use now.
