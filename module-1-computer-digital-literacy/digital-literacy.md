## ğŸ“ 4. Professional Naming Conventions
Computers are picky. How you name your files matters.

*   **No Spaces**: `my-new-page.html` is better than `my new page.html`.
*   **Lowercase Only**: Always use lowercase for web files to avoid server issues.
*   **Kebab-Case**: Separate words with dashes (`about-us.html`).
*   **Descriptive Tags**: Use clear names like `logo.png` instead of `image123.png`.
